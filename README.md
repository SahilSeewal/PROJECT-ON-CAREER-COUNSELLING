# PROJECT-ON-CAREER-COUNSELLING

1. **This project is a desktop application prototype developed using JDBC (Java Database Connectivity).** It includes the use of the following technologies:
   - Java
   - SQL

2. **JSwing (Swing Framework)** is used for the creation of the GUI, which includes buttons, text fields, labels, tables, etc.

3. **Project Objective:**
   This project aims to create an application that provides career options for different levels of qualifications (i.e., for 8th, 10th, and 12th pass students). It includes the following pages:

   i. **User Registration Page**  
      - Users who want to access this application need to register.  
      - Validation is applied on email and password.

   ii. **Login Page for Users**  
      - Users can log in after registering.

   iii. **Home Page**  
      - Contains menu items in the navbar as `8th class`, `10th class`, `12th class`, `USER`, and `ADMIN` (if the user is an admin).  
      - User/admin can sign out as well.

   iv. **Course Page**  
      - After clicking on any of the `8th class`, `10th class`, and `12th class` buttons, the user is redirected to the course page.
      - The user can select the course name from a dropdown menu and get the course details.
      - There is a direct link available to a website where the user can get more details about the course.
      - The page also contains a text area displaying all the details about the course.
      - At the bottom, there is a button to print the displayed course information.

   v. **Admin Login Page**

   vi. **Admin Home Page**  
      The admin has the following options:
      - Add/Delete a course by clicking on the `ADD`/`DELETE` buttons, respectively.
      - Create a main admin if one does not already exist by clicking on the `CREATE MAIN ADMIN BUTTON`.
      - The main admin can appoint different sub-admins.
      - The main admin can delete any sub-admin.
      - The main admin can promote any sub-admin to the position of main admin.

   vii. **Sub-Admins**  
      - Sub-admins can only add new courses and delete existing ones.
